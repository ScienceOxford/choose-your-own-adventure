Using Twine

Please check that you can access Twine from your device before the workshop.


Opening Twine

Using the device you plan to use in the workshop, visit:

twinery.org/2

You should see a screen like the below picture.

If you get a welcome message first, click 'skip'.



Starting a new story

Click the + story button on the right, and type in a title (you will be able to change this later), then click + Add.


You should now see a screen like the below picture.



Check that you can see all of the buttons on the bottom toolbar.

If you are using a tablet, or other device with a smaller screen, you may need to rotate into landscape mode, and/or scroll the toolbar to see all of the buttons.


Viewing a story

Finally, we need to check that you can view a created story!

Click the > Play button on the bottom toolbar.

In a new tab or window, you should see something like the below picture.


The story is blank because we haven't written it yet!


What now?

If you followed the steps above, and it works as expected, you do not need to do anything. You will be able to take part in the workshop as planned.


If you have any problems with Twine, please get in touch via email so we can help you to troubleshoot.
